# 🧊 Next.js AI Starter

**基于极速技术栈的现代化全栈模板。** 集成了 **Next.js 15 (App Router)** + **Tailwind 4** + **AI 能力**，内置一套惊艳的 **iOS26 液态玻璃 UI** 设计方案。

## ✨ 核心亮点

* **🧪 顶级技术栈**: 采用 **React 19 / Next.js 15 / Tailwind 4.1** —— 提前拥抱 Web 开发的未来。
* **🎨 液态玻璃 UI**: 深度定制的 `.liquid-glass-*` 组件类，实现 iOS26 风格的高级毛玻璃、渐变边框与动态光影。
* **🌍 极速 i18n**: 基于 `next-intl` 的全栈国际化方案，支持 SEO 友好路由与类型安全翻译键。
* **🔋 PWA 2.0**: 采用 `Serwist 9.0` (下一代 Service Worker 库)，提供原生级别的离线能力。
* **📡 类型安全请求**: 预设基于 `SWR 2.3` 的 HTTP 客户端，完美兼容 SSR 与 Client Components。
* **📊 性能监控**: 内置 Web Vitals 指标自动收集，助力满分 Lighthouse 体验。
